FAQ

This manual should serve as an introduction to the Scribr software for new users, and act as a referencable roadmap for experienced users.

New, Open, Save, or Quit File

To begin a new file, open a working text file, save a current working file, or quit a file, simply locate the "File" tab in the upper left menu, and select the desired action. Please note, only .txt files can be opened in the Scribr application

What are the different sections?

The different sections that each constitude their own grouping for the undo functionality include:

  • Words
  • Sentences
  • Paragraphs
  • Sections
  • Chapters

What kind of files can be opened in Scribr?

The only files that can be opened in Scribr are plain text files

How many pages can I be working on at one time?

You can only work on one document at a time.
